## Sobre

Sou Desenvolvedor Backend com experiência na construção de APIs, sistemas distribuídos e integrações corporativas utilizando Python.

Minha atuação concentra-se no desenvolvimento de soluções escaláveis, processamento assíncrono, automação de processos e aplicações baseadas em Inteligência Artificial, sempre priorizando arquitetura, qualidade de software, observabilidade e desempenho.

Tenho experiência no desenvolvimento de aplicações utilizando Django, Django REST Framework e FastAPI, além da implementação de arquiteturas orientadas a eventos com RabbitMQ e Celery, infraestrutura baseada em Docker e AWS e monitoramento de aplicações através de Prometheus, Grafana e Sentry.
